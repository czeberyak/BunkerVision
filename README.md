# BunkerVision: Industrial AI Pipeline

Система автоматического мониторинга уровня заполненности бункеров сырьем (картофель) с использованием Computer Vision (YOLOv11-cls) и паттернов отказоустойчивой архитектуры.

## Описание проекта
Проект разработан как Proof of Concept (PoC) для интеграции в производственную линию. 
Основной фокус сделан на **отказоустойчивость (Store-and-Forward)** и **работу в условиях дефицита данных** (использование статического ROI и аугментаций для синтеза датасета).

## Структура репозитория
- `src/` — Исходный код модулей (захват кадров, инференс, БД, отправка).
- `notebooks/` — R&D ноутбуки (выделение ROI, цветовой baseline, генерация синтетики).
- `monitor.py` — Главный оркестратор (точка входа для Windows Service).
- `cameras_config.json` — Конфигурация RTSP-потоков.

## Ключевые архитектурные решения
1. **Region of Interest (ROI):** Программное отсечение соседних бункеров для снижения шума.
2. **Pixel-based Baseline:** Использование HSV-масок для доказательства нелинейности визуального заполнения и обоснования применения ML.
3. **Асинхронная интеграция:** Локальный SQLite-буфер гарантирует сохранность данных при падении сети или недоступности ERP-системы.
